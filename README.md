This repository contains custom code for **Wu, Zhang, Garushyants et al. 2026 "Antiphage defence systems Druantia III and Zorya II synergise via shared DNA intermediates"**

Custom scripts are located in *./scripts/* folder:

*DruH_DruE_phylogenetic_trees.R* - main script to visualize DruE and DruH phylogenetic trees

*Alignment_Shannon_entropy_20260304.ipynb* - calculates relative entropy based on the given alignments for DruE and DruH

*DruE_habitats_from_Kim2026.R* - provides analysis of bacterial habitats for different clusters on DruE/DruH phylogenetic trees

*DruE_phylogeny_with_RecBCD.R* - this script is used to visualize RecBCD data on DruE phylogenetic tree

*DruH_DruE_select_clades.R* - supplementary script that compares clusters and create common set of clusters for Druantia III system

*DruIII_repr_genomes_neighborhoods.R* - vizualize and annotates genomic neighborhoods for Druantia III systems in representative genomes (data provided in Supplementary Table 2)

*ZorE_phylogeny.R* - custom code for vizualization of ZorE phylogeny

*./endopore/* - contains scripts to parse and analyze outputs of the ENDO-pore experiments
